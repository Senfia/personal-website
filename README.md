# Senyo Fiawornu - Personal Website

A minimalist personal website for Senyo Fiawornu, Technology Consultant & Cloud Solutions Architect.

## Tech Stack
- HTML5
- CSS3 (Custom Design System, Variables)
- Vite (Build Tool)

## Getting Started

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Run development server**
   ```bash
   npm run dev
   ```

3. **Build for production**
   ```bash
   npm run build
   ```
